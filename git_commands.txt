# Git Commands – (Part 1)

git init        → create a new repository  
git status      → show current repo state  
git add <file>  → stage a specific file  
git add .       → stage all changes  
git commit -m   → save staged changes  

git clone <url> → copy a remote repo locally  
git pull        → fetch and merge remote changes  
git push        → send local commits to remote  
git log         → view commit history  
git diff        → show file changes

# Git Commands – (Part 2)

git show            → show details of a commit  
git diff --staged   → show staged changes  
git rm <file>       → remove file from repo  
git mv <old> <new>  → rename/move file  
git log --oneline   → short commit history  

git branch          → list branches  
git branch <name>   → create new branch  
git checkout <name> → switch branch  
git merge <name>    → merge branch  
git fetch           → fetch remote changes only
